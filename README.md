API Test Automation (Python)
📌 О проекте
Проект автоматизации тестирования API, реализованный на Python с использованием Pytest.
В проекте реализован тестовый фреймворк с разделением слоёв, поддержкой конфигураций и проверкой API ответов.
Цель проекта — продемонстрировать навыки разработки автотестов и построения поддерживаемой архитектуры тестирования.

🧰 Технологии
Python
Pytest
HTTPX
Pydantic
Allure
GitHub Actions (CI/CD)

⚙️ Функциональность
Автоматизация тестирования API (GET, POST, PATCH, DELETE)
Валидация структуры ответов через Pydantic
Использование фикстур Pytest
Параметризация тестов
Позитивные и негативные сценарии
Генерация отчётов Allure

🧪 Как запустить проект
git clone https://github.com/Ivan-81105/autotests-api.git
cd autotests-api
pip install -r requirements.txt
pytest

📊 Отчёты Allure
pytest --alluredir=allure-results
allure serve allure-results

🔄 CI/CD
В проекте настроен автоматический запуск тестов через GitHub Actions при каждом push в репозиторий.

🧠 Особенности реализации
Используется модульная структура проекта
Реализовано переиспользование API клиентов
Валидация данных через Pydantic
Разделение логики тестов и работы с API

🚀 Цель проекта
Проект создан для демонстрации навыков автоматизации тестирования API и может быть использован как пример тестового фреймворка.
